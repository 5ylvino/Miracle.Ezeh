# Miracle Ezeh - Portfolio Website

A modern, responsive portfolio website for Miracle Ezeh, a Business Manager and Administrator.

## Features

- ✨ Modern, professional design
- 📱 Fully responsive (mobile, tablet, desktop)
- 🚀 Built with Next.js 14 and TypeScript
- 🎨 Styled with Tailwind CSS
- 🔍 SEO optimized with metadata, Open Graph, and structured data
- ⚡ Fast performance and optimized loading
- ♿ Accessible and user-friendly

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```
